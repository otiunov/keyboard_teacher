# keyboard_teacher

A small web app for teaching a child to recognize numbers and letters through keyboard interaction.

## Goal

Help a child learn to identify symbols by hearing a prompt, finding the matching key, and getting immediate feedback.

## Current direction

- Start with number recognition.
- Add letter recognition next.
- Extend into simple word practice after letters are familiar.
- Support English and Polish sessions.
- Use a single-page browser app with both on-screen and physical keyboard input.

## Repo status

This repository is currently being set up. The product and implementation are still being defined.
